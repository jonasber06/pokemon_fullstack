# pokemon_fullstack
I dette prosjektet var målet å lage en kamp-simulator for Pokemon, som fra den populære videospillserien

Prosjektet ble en omfattende fullstack-oppgave, der både HTML, CSS og Javascript måtte brukes.

I tillegg, for å hente informasjon om faktiske pokemon til spillet, ble det hentet data fra nettstedet https://pokeapi.co/

Bakgrunnsbilder er egenproduserte, 

Jeg (Jonas Bergersen) har ikke stått for all utvikling i dette prosjektet, men har utviklet JS-filen der det hentes informasjon om pokemon-figurene
som skal brukes i spillet (js_init.js), samt og noe i spillets JS-fil (finalSpill_js.js).

Spillet er utviklet vinteren 2023/2024 i første omgang, før jeg (Jonas B) gjennomførte revidering vinteren 2025/2026
