# pokemon_fullstack
I dette prosjektet var målet å lage en kamp-simulator for Pokemon, som fra den populære videospillserien

Prosjektet ble gjort med full stack, der både HTML, CSS og Javascript måtte brukes.

I tillegg, for å hente informasjon om faktiske pokemon til spillet, ble det hentet data fra nettstedet https://pokeapi.co/

Jeg (Jonas Bergersen) har ikke stått for all utvikling i dette prosjektet, men har jobbet mest med JS-filen der det hentes informasjon om pokemon-figurene
som skal brukes i spillet (js_init.js), samt og noe i spillets JS-fil (finalSpill_js.js).

Spillet er påbegyt som en del av en skoleoppgave i 2024, men er videreutviklet senere etter dette.
